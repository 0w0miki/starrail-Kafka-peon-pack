# Starrail Kafka peon pack

崩坏星穹铁道-卡芙卡 Open Peon 语音包

## 收录语音

| Event | Examples |
|---|---|
| Session starts | **"嗨，又见面啦"**, **"嗨，星穹列车我们携手作战的机会，看来不少呢"** |
| Session ends | **"该说再见了…"**, **"美妙的时光总有尽头"**, **"拜拜，下一次试着让我意外吧"** |
| Task acknowledge | **"很快就好"**, **"索性都干掉吧~"**, **"来得好"**, **"那么从现在开始行动由我接手"** |
| Task complete | **"能用"**, **"做得不错"**, **"毫不意外"**, **"这一次的剧本也顺利完成了"** |
| Task error | **"有点意思"**, **"这还…不是结局"**, **"有些难办"**, **"嗯？看他们蠢蠢欲动的样子……"** |
| Input required | **"该动手啦"**, **"剧本上没写的，我们决定就好"** |
| Resource limit | **"停下来"**, **"放轻松"** |
| User spam | **"欲速则不达"** |

*声音主要来自wiki中游戏语音，部分来自剧情语音*
